# fart.css

CSS library of styles reusable across FartLabs frontends.

## Usage

Import `fart.css` in the `<head>` of your HTML document using the JSDelivr CDN.

```html
<link
  rel="stylesheet"
  type="text/css"
  href="https://cdn.jsdelivr.net/gh/FartLabs/fart.css/fart.css"
/>
```
